#### 1.4 設定ファイル作成

`config.py` を作成:

```python
# config.py
import os

class Config:
    """アプリケーション設定"""
    
    # Flask設定
    SECRET_KEY = os.environ.get('SECRET_KEY') or 'dev-secret-key-change-in-production'
    
    # データベース設定
    DATABASE_PATH = 'database/employees.db'
    
    # ログ設定
    LOG_DIR = 'logs'
    LOG_FILE = os.path.join(LOG_DIR, 'app.log')
    LOG_LEVEL = os.environ.get('LOG_LEVEL', 'INFO').upper()
    
    # アプリケーション設定
    APP_NAME = '社員情報管理システム'
    VERSION = '1.0.0'
```

```bash
EOF
```

## 付録A: SQLAlchemy ORM への移行手順

> **対象ステップ**: ステップ3（データベース管理実装）の sqlite3 実装を SQLAlchemy ORM に移行する場合の手順です。
> 詳細は **詳細設計書.md → D-01-B** を参照してください。

### A-1. SQLAlchemy インストール

```bash
pip install sqlalchemy
pip freeze > requirements.txt
```

### A-2. モデルファイル作成

`database/models.py` を新規作成し、テーブル定義をPythonクラスとして記述します。

```
参照: 詳細設計書.md → D-01-B → 新規ファイル: database/models.py
```

**ファイル作成**:
```bash
touch database/models.py
```

詳細設計書.md D-01-B の「新規ファイル: `database/models.py`」コードを実装してください。

### A-3. database_manager.py の書き換え

`database/database_manager.py` を SQLAlchemy を使った実装に書き換えます。

```
参照: 詳細設計書.md → D-01-B → 変更後: database/database_manager.py
```

**変更点の要約**:

| 変更前（sqlite3） | 変更後（SQLAlchemy） |
|-----------------|-------------------|
| `import sqlite3` | `from sqlalchemy import create_engine` |
| `sqlite3.connect()` | `create_engine(f"sqlite:///{db_path}")` |
| `cursor.execute(CREATE TABLE SQL)` | `Base.metadata.create_all(engine)` |
| `get_connection()` → `sqlite3.Connection` を返す | `get_session()` → `Session` を返す |

### A-4. 動作確認

```bash
# データベース初期化テスト
python3 -m database.database_manager

# テーブル・インデックスが作成されていることを確認
sqlite3 database/employees.db << 'EOF'
.schema employees
.indexes employees
.quit
EOF
```

**✅ チェックポイント**:
- [ ] `sqlalchemy` が `requirements.txt` に含まれている
- [ ] `database/models.py` が作成されている
- [ ] `database/employees.db` が作成されている
- [ ] `employees` テーブルと3つのインデックスが存在する

### A-5. ルート側のDB呼び出し変更

`get_connection()` を使っている箇所を `get_session()` に変更します。

```python
# 変更前（sqlite3）
conn = db_manager.get_connection()
cursor = conn.cursor()
cursor.execute("SELECT * FROM employees")
rows = [dict(r) for r in cursor.fetchall()]
conn.close()

# 変更後（SQLAlchemy ORM）
from database.models import Employee

with db_manager.get_session() as session:
    employees = session.query(Employee).all()
    # emp.name, emp.department のようにドット記法でアクセス
```
